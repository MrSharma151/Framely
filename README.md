# Framely  

Framely is an **optical e-commerce web application** designed to manage and sell eyewear products like glasses, sunglasses, and contact lenses.  

---

## ✅ Current Project Status  

- **Backend (ASP.NET Core Web API)**  
  - ✔ Product, Category, and Order APIs created  
  - ✔ DTOs & AutoMapper integrated  
  - ✔ JWT Authentication + Role-based Authorization implemented  
  - ✔ Pagination, Sorting, Filtering, and Searching for products  
  - ✔ Basic validations & error handling added  
  - ✔ EF Core with migrations applied  
  - ✔ API tested with Swagger & partially verified  

  **Areas of improvement:**  
  - Advanced validations  
  - Better error standardization  
  - Unit & integration testing  
  - Response wrapping for consistency  

- **Frontend (Next.js + Tailwind)**  
  - ✔ Initial Next.js setup created  
  - ✔ Tailwind CSS configured  
  - ✔ Basic clean folder structure ready  
  - ⏳ Further UI development pending  

---

## 🔄 In Progress

## 🛠 Tech Stack
